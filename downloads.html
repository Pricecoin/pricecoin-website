Troubleshooting PricecoinX:

If your node fails to connect to any nodes on start, please go to Help ‣ Debug ‣ Window ‣ Console and enter the following commands:

  • addnode 51.158.121.4 add
  • addnode 51.15.215.166 add
  • addnode 107.112.216.59 add
  • addnode 79.177.26.188 add

If it's still not working, try these ones too:

  • addnode 216.106.156.204 add
  • addnode 95.179.47.203 add
  • addnode 2.133.0.154 add
  • addnode 91.105.181.86 add
  • addnode 217.182.166.152 add
  • addnode 217.182.166.152 add
  • addnode 104.37.175.112 add

If you're still having trouble, please contact us on Discord.
